# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 6.5.x   | ✅ Supported |
| 6.0.x   | ✅ Supported |
| 5.0.x   | ❌ Not supported |
| < 5.0   | ❌ Not supported |

## Reporting a Vulnerability

If you discover a security vulnerability in this script, please report it by creating an issue on GitHub with the "security" label.

**Please do not disclose security vulnerabilities publicly until they have been addressed.**

When reporting, please include:

1. Description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Suggested fix (if any)

We will respond to security issues within 48 hours and work on a fix as quickly as possible.
